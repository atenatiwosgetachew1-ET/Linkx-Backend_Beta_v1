Degree Centrality
Measures how many direct connections a node has. Nodes with high degree serve as hubs that connect multiple regions. It reflects raw connectivity — the foundation of network structure.
Betweenness Centrality
Identifies nodes that act as bridges between parts of the network. High-betweenness nodes influence communication flow and control paths. They’re key intermediaries, brokers, and strategic connectors.
Closeness Centrality
Reflects how near a node is to all others. High closeness means a node can spread information quickly. It’s ideal for detecting efficient communicators and influencers.
Eigenvector Centrality
Measures not just how many connections a node has, but how influential its neighbors are. Nodes linked to other powerful nodes score highly here.
PageRank
Calculates importance based on inbound links from other nodes. A node gains weight when important nodes connect to it. The same principle powers modern web search algorithms.
